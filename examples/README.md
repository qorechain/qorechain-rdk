# Examples

Short, focused, runnable examples — one per capability:

- connect and read module parameters
- suggest a profile from a use-case string
- create a rollup from each preset
- override a custom config and validate it
- submit a settlement batch (each settlement path)
- challenge and resolve an optimistic batch
- submit and query a native DA blob
- list rollups and decode lifecycle events
- read rollup / batch status via the `qor_` JSON-RPC namespace

**Status: coming soon.** Each example will have its own README and minimal
dependencies, and run against the `qorechain-diana` testnet.
